Özellikler
Neler yapabilir?
💽

Çoklu Disk Tarama

Windows'taki tüm aktif sürücüleri (C:\, D:\, E:\…) otomatik olarak algılar ve paralel şekilde tarar.

📂

Proje Gruplandırma

requirements.txt, package.json, .git gibi işaretçileri kullanarak dosyaları proje klasörlerine atar.

🔍

Arama & Filtre

Raporda proje adı, dosya adı veya dile göre anlık arama ve filtreleme yapılabilir.

👁

Kod Önizleme

Dosyaya tıklandığında ilk 6 satır rapor içinde görüntülenir — editör açmaya gerek yok.

📊

Dil İstatistikleri

Her projedeki dil dağılımı renk kodlu rozetlerle gösterilir; genel istatistik üst barda özetlenir.

📄

Bağımsız HTML

Çıktı tek bir HTML dosyasıdır; sunucu gerektirmez, tarayıcıda çevrimdışı açılır.

Nasıl başlarım?
1

Repoyu klonla

git clone https://github.com/Eren-Oztk/Kod_Tarama.git komutuyla indir. Python 3.8+ gereklidir; dış bağımlılık yoktur.

2

Çalıştır

python kod_tara.py — veya Windows'ta TARA.bat dosyasına çift tıkla. Tarama 1-5 dakika sürebilir.

3

Raporu aç

Tarama bittiğinde masaüstünde kod_haritasi.html otomatik açılır. Tarayıcıda çevrimdışı çalışır.
